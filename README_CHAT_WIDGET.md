# 悬浮智能聊天框（免费/低代码）接入说明

本项目是 Flask 静态站点形态，最稳妥的“免费 + 可快速上线 + 带知识库”的方式，一般是 **嵌入第三方的免费客服/知识库聊天组件**（零代码）。

同时，为了避免第三方服务受限，我们也提供一套 **项目内置的低代码 FAQ/文档检索机器人**：
- 完全本地，不调用任何付费大模型接口
- 支持上传 FAQ/文档（Markdown/TXT）
- 只回答常见问题；无法命中则提示“转人工客服”（满足你的第 2 条）

> 说明：
> - 你提到的“豆包稳定免费的 AI 接口”目前官方并不存在一个**长期稳定且完全免费的**公开大模型 API（大模型通用接口多数都计费或有额度限制）。
> - 因此我给你两个落地选项：
>   1) **真正零代码的第三方嵌入（如果你选定具体平台，我可以把代码补齐）**
>   2) **仓库内置的本地知识库机器人（已实现）**

---

## A. 零代码/低代码方案（推荐上线最快）

你只需要选择一个支持「知识库 + 网站嵌入」的平台，然后把它给你的 `<script>` 粘到 `templates/base.html` 末尾即可。

常见平台通常会提供：
- 知识库（上传 PDF/Doc/FAQ）
- 机器人优先回答，转人工
- 悬浮按钮样式、欢迎语配置

**你定平台后**（比如你已有账号），把平台的嵌入脚本发我，我可以：
- 帮你放到正确的位置
- 避免和 Bootstrap/CSS 冲突
- 做移动端适配

---

## B. 本项目内置：FAQ/文档知识库机器人（完全免费、本地运行）

### 1) 功能
- 悬浮聊天按钮 + 右下角聊天窗口
- 使用 `kb/` 目录下的 FAQ/文档回答
- 匹配不到：提示“问题较复杂，建议转人工”，并给出联系入口
- 支持自定义：图标/欢迎语/人工联系方式

### 2) 配置知识库
把你的资料以以下任意格式放到：
- `kb/faq.json`（结构化 FAQ）
- `kb/docs/*.md` / `kb/docs/*.txt`（文档）

更新后无需改代码，重启 Flask 即可。

### 3) 运行
启动 Flask：

```bash
python3 app.py
```

打开：
- 首页右下角会出现悬浮聊天

## 可复制的嵌入代码（如果你要粘到其它站）

把下面这段粘到页面 `</body>` 前即可：

```html
<link rel="stylesheet" href="/static/css/chat_widget.css">
<div id="sl-chat" data-welcome="您好，我是留学小助手，有什么问题都可以咨询我哦！" data-human-url="/contact"></div>
<script src="/static/js/chat_widget.js"></script>
```

> 如果不是本项目，需要把 `/static/...` 改成你站点对应的静态资源路径。

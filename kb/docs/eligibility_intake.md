# 选校/能否申请：信息采集与判断口径（本科/修士）

> 用途：当学生描述了很多背景，询问“能不能申某个学校/某个研究科/某个项目”时，先用统一口径做信息采集，再给出可判断的结论或下一步。

## 1. 常见问题类型
- 我这种背景能申请 XXX 大学/研究科/专业吗？
- 我想申请修士/研究生（旁听生），需要达到什么条件？
- 我成绩一般/跨专业/无科研/无日语，能买吗？
- 我想冲刺名校/国公立，有机会吗？

## 2. “是否能申请”需要的关键变量（先问清楚）

### 2.1 目标信息
- 目标学校（中文/日文名都可以）
- 目标研究科/专业方向（尽量具体）
- 入学时间：2026年4月/10月？还是先研究生再修士？

### 2.2 学术背景
- 本科院校/专业
- GPA 或均分（以及是否有挂科/重修）
- 毕业时间
- 相关课程/作品集（设计/艺术/建筑等）

### 2.3 语言成绩（修士/研究生判断重点）
> 多数日本修士非常看：日语（JLPT）与英语（托福/雅思/托业），具体看研究科与导师要求。

请明确：
- 日语：JLPT 等级与分数（N1/N2/N3；若无成绩，当前学习进度）
- 英语：TOEFL/IELTS/TOEIC 分数（若无成绩，计划考试时间）

### 2.4 研究相关（修士/研究生判断重点）
- 意向研究方向（2-3句话即可）
- 是否有科研/论文/竞赛/项目经历
- 是否已准备研究计划书（大纲也可以）

### 2.5 其他因素
- 预算区间（国公立/私立差异大）
- 是否能去日本参加校内考/面试
- 是否需要奖学金

## 3. 快速判断口径（谨慎表述）

### 3.1 可以初步判断的情况
- 学校/研究科要求明确、学生语言与背景已满足：可以给出“可申请/建议冲刺/建议保底”的方向性建议。
- 如果学生缺少关键语言成绩：先引导补齐语言信息，再判断。

### 3.2 必须转人工进一步评估的情况
- 目标为名校/热门研究科且背景复杂（跨专业、GPA偏低、无语言、时间紧）
- 学生想同时评估多个学校/多个方向
- 学生需要给出更细致的选校梯度与时间线

## 4. 推荐的“追问模板”（可直接复制）

你可以先把以下信息发我，我再帮你判断更准确：
1）目标学校/研究科（或你想去的城市/方向）
2）入学时间：打算 2026年4月/10月？还是先研究生再修士？
3）本科背景：学校+专业+GPA/均分
4）语言成绩：JLPT（N几/分数）+ TOEFL/IELTS/TOEIC（分数）
5）研究方向与经历：想研究什么？是否有科研/项目/论文？

如果你暂时没有语言成绩，也可以告诉我目前水平与预计考试时间，我们可以按目标倒推备考与申请节奏。

## 5. 日语/英语成绩不足时的引导话术
- “很多研究科会要求 JLPT N2 或 N1（也有看面试/校内考表现的），英语项目则更看托福/雅思。你目前的日语/英语成绩分别是多少（或预计什么时候能考出来）？”
- “为了判断你能否申到目标学校，需要先确认语言门槛。你方便告诉我：JLPT N几？托福/雅思/托业多少分吗？”

## 6. 输出结论的安全话术（避免绝对承诺）
- “从你目前信息看，有申请空间，但最终要以学校/研究科当年募集要项为准。”
- “建议我们把语言与研究方向再明确一下，我可以帮你做一个冲刺/匹配/保底梯度。”
- “如果你愿意，把目标学校+语言成绩发我，我可以给你一个更具体的可行性判断与下一步准备清单。”
